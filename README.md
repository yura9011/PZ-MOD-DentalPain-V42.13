# Dental Pain

Adds a realistic dental health system to Project Zomboid. Brush, floss, and maintain oral hygiene or suffer decay, toothaches, and eventual tooth loss.

## Features

- **Dental Hygiene System:** Manage your oral health using toothbrushes, toothpaste, floss, and mouthwash.
- **Pain Mechanics:** Neglecting your teeth leads to dental pain, which affects your character's performance and mood.
- **Medical Procedures:** Perform extractions when a tooth is beyond saving (requires proper tools and anesthetic).
- **Crafting:** Recipes for homemade dental supplies.
- **Bilingual Support:** Full translations in English and Spanish.
- **Build 42 Compatible:** Specifically structured for the latest game updates.

## Planned Features

- Custom icons for all dental items.
- Expanded food damage table.
- Specific animations for dental actions.
